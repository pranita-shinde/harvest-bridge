# 🌾 Harvest Bridge

A full-stack digital platform designed to empower farmers with smart agricultural tools, crop recommendations, and direct marketplace connectivity.

---

## 🚀 Features

- 🌱 Crop Prediction using Machine Learning  
- 🧪 Fertilizer & Pesticide Recommendation  
- 🤖 Chatbot Assistance for farmers  
- 🛒 Farmer-to-consumer marketplace  
- 📊 Data-driven decision support  
- 🔐 User authentication & profile management  

---

## 🏗️ Project Structure
harvest-bridge/ ├── mainPage/ # React frontend (UI/UX) ├── cropCare/ # ML models & Flask backend ├── farm-app/ # Full-stack app (backend + frontend) ├── farmConnect/ # Additional frontend modules

### Frontend
- React.js

## ▶️ How to Run

### 1️⃣ Frontend (mainPage)
cd mainPage npm install npm start

### 2️⃣ ML Backend (cropCare)
cd cropCare pip install -r requirements.txt python app.py

### 3️⃣ Farm App
cd farm-app
